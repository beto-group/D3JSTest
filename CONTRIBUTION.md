# Contributing to D3.JS TEST

We welcome contributions to improve this component!

## Contribution Guidelines

1.  **Scope**: Ensure your enhancements align with the Datacore component standards (e.g., synchronous View patterns, zero external dependencies, robust resource caching).
2.  **Aesthetics**: Follow the "Black on Black on Black" theme, using HSL-tailored accents, smooth animations, and glassmorphism.
3.  **Performance**: Always ensure animations use proper cleanup handlers (e.g., cancelling `requestAnimationFrame`) to prevent memory leaks.
4.  **Testing**: Verify functionality inside the Datacore environment before submitting modifications.
